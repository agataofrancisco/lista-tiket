# lista-tiket

um projecto com o objectivo de dinamizar a reserva de bilhetes para eventos infantis. a ideia é que o adulto se cadastre informe quantas crianças vai levar e as suas idades. se elas estiverem abaixo de 5 anos, não pagam, mas de 5 para cima pagam o bilhete, cada bilhete custa 1000kz. o adulto deverá colocar o seu nome, numero de telefone e email. quando terminar de colocar as informações o site fará a soma das pessoas que vão, se incluirem crianças que não pagam, não deverá incluir na conta, deve apresentar o metodo de pagamento via api da appypay, seja multicaixa express ou codigo qr. quando a pessoa pagar, o pagamento for confirmado, o site ira conversar com o google forms e irá colocar as informações do adulto, o numero de crianças, as idades e numero de bilhetes que a pessoa tem direito. ao mesmo tempo tera de criar um codigo qr com o nome, e o numero de bilhetes que a pessoa tem acesso para ser comprovado no dia do evento
